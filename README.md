# 🇮🇳 Unfold India

> Discover India's hidden gems with AI-powered travel planning.

Unfold India is a modern travel application designed to help users explore India's famous landmarks, hidden destinations, and local experiences. The app combines AI-powered recommendations, interactive maps, route planning, and a travel assistant to make trip planning simple and enjoyable.

## ✨ Features

- 🗺️ Discover popular attractions and hidden gems
- 🤖 AI-powered travel recommendations
- 📍 Interactive maps and navigation
- 🛣️ Smart route planning
- 💬 AI travel assistant/chatbot
- 🔍 Search destinations across India
- ⭐ Save favorite places
- 🌤️ Weather information
- 📱 Clean and intuitive UI

## 🛠️ Tech Stack

### Frontend
- React Native
- Expo
- TypeScript

### Backend
- FastAPI
- Python

### APIs
- Geoapify API
- OpenStreetMap
- AI Chat Integration

## 🚀 Getting Started

```bash
git clone https://github.com/Gireesh-mlgs/UnfoldIndia-App.git
cd UnfoldIndia-App
npm install
npx expo start
```
